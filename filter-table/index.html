🔖 FilterTable ブックマークレット

Wikiru.jpのフィルタテーブルを拡張し、行の自由な並べ替え・OR検索とAND検索入れ替えを行えます。設定はキャッシュに保存できます。

📋 使い方

  1. 下のボタンをドラッグしてブックマークバーにドロップ
  2. または、ページのブックマークを作成して、URLを下のコードに置き換える
  3. フィルタリング機能が必要なページで、ブックマークをクリック
  4. 表のヘッダーや行操作ボタンが表示されます
